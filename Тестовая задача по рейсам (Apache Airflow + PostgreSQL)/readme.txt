Тестовое задание 
1. Скачать набор данных о воздушном движении OpenSky 
2. Разбить скачанный файл на блоки (chunks) 
2. Написать даг в AirFlow* для загрузки данных в PostgreSQL*. 
НЕ должены загружаться рейсы, у которых не указан origin или destination, также НЕ должны загружаться рейсы, у которых origin и destination совпадают 
3. Сформировать SQL-запрос: вывести callsign, registration, количество рейсов за всё время для callsign = ETD18U с разбивкой по каждому из registration (бортовому номеру самолета) 
 
Что ожидается в качестве результата выполнения задания: 
1. Код AirFlow-дага из п.2 тестового задания 
2. Код SQL формирования таблицы в PostgreSQL из п.2 тестового задания 
3. Код SQL-запроса из п.3 тестового задания
